About RPG Tsukuru 2003

RPG Tsukuru 2003/RPG Maker 2003/RPG Maker 2K3 is a game engine based in events released your first version(1.00) in 2002. It based in events so you cannot to access some functions like put image before the title screen like XP and VX(Vista Experience).


In this engine you can make advance systems if you have learned how to program a little bit. It's a good engine if you want to developer game like Super Famicom or Mega Drive like Final Fantasy, Dragon Quest, Phantasy Star and Zelda. On menu in the left (Download) you can get the backup from original first version from this engine. Please read the instructions before mount and install
